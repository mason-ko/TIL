# Step 4: GraphRAG 기본

## 개요
벡터 검색과 그래프 탐색을 결합하여 더 강력한 RAG 시스템을 만듭니다.

## GraphRAG의 핵심 아이디어

**벡터 검색:** 의미적 유사성
**그래프 탐색:** 구조적 관계

→ 둘을 결합하면 더 정확하고 풍부한 컨텍스트!

## 하이브리드 접근

1. 벡터 검색으로 관련 엔티티 찾기
2. 그래프에서 관계 확장
3. 통합된 컨텍스트로 답변 생성

## 다중 홉 추론

그래프를 따라 여러 단계 추론:
```
Alice → CompanyA → ProductX → Python
```

질문: "Alice가 사용하는 언어는?"
답변: 그래프 경로를 따라 Python 도출
