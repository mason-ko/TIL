# Step 1: 기본 멀티 에이전트 - 순차 실행

## 목표

- 멀티 에이전트 개념 이해
- 순차 실행 패턴 구현
- 에이전트 간 상태 전달

## 왜 멀티 에이전트?

```python
# 단일 에이전트 (복잡함)
Agent: 검색도 하고, 분석도 하고, 요약도 하고...
→ 프롬프트 길어짐
→ 성능 저하

# 멀티 에이전트 (전문화)
Researcher: 검색 전문
Analyzer: 분석 전문
Writer: 요약 전문
→ 각자 최적화
→ 높은 품질
```

## 기본 패턴: Sequential

```python
Agent1 (검색) → Agent2 (분석) → Agent3 (요약)
```

## 핵심 요약

- **멀티 에이전트 = 전문 에이전트 협업**
- **순차/계층/네트워크 패턴**
- **복잡한 작업 분할**
