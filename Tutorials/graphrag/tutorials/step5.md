# Step 5: 커뮤니티 감지와 요약

## 개요
큰 그래프를 의미 있는 클러스터(커뮤니티)로 나누고 각각을 요약합니다.
이는 Microsoft GraphRAG의 핵심 아이디어입니다.

## 커뮤니티 감지

그래프의 밀집된 부분을 찾아 그룹화:
- AI/ML 관련 노드들 → 하나의 커뮤니티
- 프로그래밍 언어들 → 다른 커뮤니티

## 계층적 요약

각 커뮤니티를 LLM으로 요약:
```
커뮤니티 1 → 요약 1
커뮤니티 2 → 요약 2
...
전체 → 최종 요약
```

## 전역 vs 지역 검색

**전역 검색:** 커뮤니티 요약 사용 (넓은 질문)
**지역 검색:** 상세 문서 사용 (구체적 질문)
